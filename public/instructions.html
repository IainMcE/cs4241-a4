Liars Dice - Game Instructions

Rules of the Game

Liars Dice is a popular dice game that combines strategy and bluffing. The game is played with dice and can have multiple players. Here are the basic rules:

  1. Each player starts with a set of dice (typically 5 dice per player).
  2. Players take turns making bids on the total number of dice showing a specific value (e.g., "Three 4s").
  3. Players can either raise the bid or challenge the previous bid.
  4. If challenged, all dice are revealed, and if the bid was accurate, the challenger loses a die. If the bid was inaccurate, the bidder loses a die.
  5. The game continues until only one player has dice remaining, and that player wins.

How to Use the UI

To enjoy a game of Liars Dice, follow these steps using the provided user interface (UI):

  1. Start a New Game: Click the "Re-roll" button in the main menu to begin a new game of Liars Dice.
  2. Make Bids: During your turn, use the "Place your Call" section to input your bid. Specify the number of dice and their value (e.g., "Three 4s").
  3. Submit Your Bid: After entering your bid, click the "Place Call" button to submit it to the game.
  4. Challenge or Raise: Once a bid is submitted, you can choose to challenge the previous bid or raise it by specifying a higher quantity or value.
  5. Game Progress: Keep track of the game's progress on the screen, including the number of dice each player has and the current bid.
  6. Winning: The game continues until only one player has dice left. That player is declared the winner.
